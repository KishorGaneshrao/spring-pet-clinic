# Spring Pet Clinic 
This is a sample Spring Boot Application that demonstrates the use of Spring Boot, Spring Data JPA, Flyway, Validation, and Spring Web.
The application is a simple pet clinic management system that allows users to manage pets, their owners, and visits.
